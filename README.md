A complete Machine Learning Operations (MLOps) pipeline demonstrating the automated lifecycle of an ML model. The project covers local training on biomedical time-series data (MIT-BIH Arrhythmia Database), cloud deployment via Azure Machine Learning, production traffic simulation, request logging to Azure SQL, and performance/data drift monitoring with PowerBI.

Tech Stack:

*ML Framework: Python, TensorFlow / Keras

*Cloud & MLOps: Azure Machine Learning (Model Registry, Endpoints)

*Database: Azure SQL Database

*Monitoring: PowerBI Desktop
